## 現在のプロトタイプ

`npm run dev` で動くプロトタイプがあります。Tailwind CSS で最低限のスタイルをつけた状態。

ページ構成：
- `/` トップページ
- `/universities` 大学一覧（検索・フィルタ付き）
- `/universities/[slug]` 大学詳細
- `/about` このサイトについて
