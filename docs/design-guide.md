## 1. サイトの方向性

### トーン・雰囲気

- **信頼感**: 公的な情報を扱うサイトとして、落ち着いた印象
- **アクセシブル**: 誰にとっても使いやすく、見やすい
- **ニュートラル**: 特定の大学を推奨・批判しない中立的な姿勢

> **要相談**: レインボーカラーを使うか？使うならどの程度か？

## 2. ページ構成

現在のプロトタイプの画面構成：

```
/                    トップページ
/universities        大学一覧（検索・フィルタ付き）
/universities/[slug] 大学詳細
/about               このサイトについて
```

### 各ページの役割

| ページ | 主な機能 | 備考 |
|---|---|---|
| トップ | サイト説明、大学一覧への導線、統計 | ファーストビューが重要 |
| 大学一覧 | カード一覧、検索、フィルタ | メインのページ |
| 大学詳細 | 施策の表形式表示、出典リンク | 情報量が多いのでレイアウト重要 |
| About | サイトの説明、注意事項 | |
