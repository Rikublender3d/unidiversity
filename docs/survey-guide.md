## 2. 判定基準

各項目は以下の4段階で評価します：

| 記号 | 意味 | 基準（案） |
|---|---|---|
| ○ | 対応あり | 制度・仕組みが明確に存在し、公開情報で確認できる |
| △ | 一部対応 | 部分的に対応、または検討中・試行段階 |
| × | 未対応 | 対応がない、または確認できない |
| 不明 | 情報なし | 公開情報から判断できない |

> **要相談**: この基準で良いか？もっと細かくするか？○×だけにシンプルにするか？

## 3. 調査の進め方

### 情報ソース

1. 大学公式サイト
2. ダイバーシティ推進室等の公開資料
3. 大学が出しているガイドライン・報告書
4. ※ SNSや個人ブログは原則ソースにしない

### 調査フロー（案）

1. 対象大学リストを決める
2. 各大学のサイトから上記項目を調査
3. スプレッドシート等に記入
4. レビュー（まささん）
5. JSON/CMSへの反映（りく）
