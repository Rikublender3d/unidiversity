# UniDiversity 要件定義書

> **ステータス: ドラフト（たたき台）**
> チーム全員で相談しながら更新していくドキュメントです。

## 1. プロジェクト概要

全国の大学におけるダイバーシティ推進（特にLGBTQ関連施策）の取り組みを一元的に閲覧・比較できる公開Webサイト。

### 背景

- 大学のダイバーシティ推進状況はバラバラに公開されており、横断的に比較する手段がない
- 比較サイトがあれば、各大学の施策推進を促す効果が期待される
- Wiki的に年1回程度、みんなで更新していく運用を想定

### サイト名

- **UniDiversity**（仮）
- → チームで要相談

---

## 2. ターゲットユーザー

| ユーザー | ニーズ |
|---|---|
| 大学生（特にLGBTQ当事者・Ally） | 進学先・編入先の比較、自大学の改善提案の根拠 |
| 大学教職員・ダイバーシティ推進担当 | 他大学のベストプラクティス参照 |
| 研究者・ジャーナリスト | 施策の全体像把握、分析・報道 |

---

## 3. 機能要件

### Phase 1（MVP）

- **トップページ**: サイト概要、大学一覧への導線、統計情報
- **大学一覧**: カード一覧、検索、地域/種別フィルタ
- **大学詳細**: 各施策カテゴリの表示、出典リンク
- **このサイトについて**: 目的・注意事項の説明

### Phase 2（将来拡張）

- 複数大学の横並び比較機能
- 施策充実度のスコアリング・ランキング
- 地図ビュー（日本地図上にプロット）
- データ更新リクエスト機能（フォーム送信）
- CSVエクスポート

---

## 4. 非機能要件

| 項目 | 要件 |
|---|---|
| レスポンシブ | モバイル・タブレット・PCに対応 |
| アクセシビリティ | WCAG 2.1 AA準拠を目標 |
| SEO | メタタグ、OGP対応 |
| 多言語 | Phase 1は日本語のみ |

---

## 5. 運用方針

- データ更新は年1回程度を想定（年度末〜年度初め）
- 情報の正確性は出典URLの明記で担保
- GitHub上でのIssue/PRベースでのフィードバック受付
